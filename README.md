## 安装指南

### 基础安装（普通用户）

1. 确保已安装Python 3.6或更高版本
2. 打开命令提示符/终端，输入以下命令：
   ```
   pip install pillow matplotlib numpy svgwrite cairosvg
   ```
3. 下载程序文件
4. 双击运行`Icon_Generator_Tool.py`

### 高级安装（开发者）

1. 克隆GitHub仓库：
   ```
   git clone https://github.com/duma520/icon-generator-tool.git
   ```
2. 创建虚拟环境：
   ```
   python -m venv venv
   ```
3. 激活虚拟环境：
   - Windows: `venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. 安装依赖：
   ```
   pip install -r requirements.txt
   ```

### 系统要求

- **最低配置**：
  - 操作系统：Windows 7/10, macOS 10.13+, Linux
  - 内存：2GB
  - 存储空间：100MB

- **推荐配置**：
  - 操作系统：Windows 10/11, macOS 11+, Ubuntu 20.04+
  - 内存：4GB+
  - 存储空间：200MB

## 基础使用教程

### 1. 图片转图标（适合初学者）

**步骤1**：打开软件，选择"图片转图标"标签页

**步骤2**：点击"浏览"按钮选择图片

**步骤3**：调整基本参数：
- 亮度：控制图标明暗
- 对比度：增强或减弱对比
- 饱和度：调整颜色鲜艳程度

**步骤4**：选择形状（方形、圆形、心形等）

**步骤5**：设置尺寸（如16x16, 32x32等）

**步骤6**：点击"生成预览"查看效果

**步骤7**：满意后点击"保存图标"

**示例**：将公司logo转换为圆形图标，适用于网站favicon

### 2. 文字转图标（适合办公人员）

**步骤1**：选择"文字转图标"标签页

**步骤2**：输入文字内容（如公司缩写）

**步骤3**：选择字体和颜色

**步骤4**：设置背景样式（纯色/渐变/透明）

**步骤5**：生成并保存

**示例**：制作"VIP"文字图标用于会员系统

## 专业功能详解

### 1. 高级图像处理（适合设计师）

- **效果滤镜**：
  - 模糊：创建柔和效果
  - 浮雕：添加3D质感
  - 边缘增强：突出轮廓

- **专业调整**：
  ```python
  # 代码示例：应用棕褐色滤镜
  def apply_sepia(img):
      width, height = img.size
      pixels = img.load()
      for py in range(height):
          for px in range(width):
              r, g, b = img.getpixel((px, py))[:3]
              tr = int(0.393 * r + 0.769 * g + 0.189 * b)
              tg = int(0.349 * r + 0.686 * g + 0.168 * b)
              tb = int(0.272 * r + 0.534 * g + 0.131 * b)
              pixels[px, py] = (min(255, tr), min(255, tg), min(255, tb))
      return img
  ```

## 常见问题解答

### 基础问题

**Q**：生成的图标模糊怎么办？
**A**：1. 检查原始图片分辨率 2. 提高输出质量设置 3. 尝试锐化滤镜

**Q**：如何制作透明背景图标？
**A**：在背景设置中选择"透明"选项

### 技术问题

**Q**：SVG转换失败怎么处理？
**A**：1. 检查SVG代码语法 2. 尝试简化复杂路径 3. 使用示例代码测试

**Q**：Emoji显示为方框怎么办？
**A**：安装Emoji字体包（如Segoe UI Emoji）

## 技术原理

### 核心架构

1. **图像处理引擎**：基于Pillow库
2. **矢量图形处理**：SVGWRITE + CairoSVG
3. **多线程处理**：Python threading模块

### 关键算法

- 图像重采样：Lanczos算法
- 颜色空间转换：RGB/HSV/HSL
- 形状蒙版：Alpha通道合成
